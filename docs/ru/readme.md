# Документация teleflow

teleflow превращает аннотированные Python-датаклассы в полноценные Telegram-интерфейсы — пошаговые визарды, списки с пагинацией, панели настроек и многое другое. Вы описываете, что должен делать бот, а teleflow генерирует клавиатуры, пагинацию, сессии и состояние.

Этот гайд проведёт вас по библиотеке концепция за концепцией. Если вы новичок — начните сверху.

## Порядок чтения

1. **[Начало работы](getting-started.md)** — установка, первый бот, цикл компиляции
2. **[Flows и виджеты](flows.md)** — основной паттерн: многошаговые диалоги и 30+ виджетов
3. **[Представления: Browse, Dashboard и Search](views.md)** — отображение данных и взаимодействие с ними
4. **[Settings](settings.md)** — пользовательские настройки с инлайн-редактированием (переиспользует виджеты flow)
5. **[Трансформы](transforms.md)** — отмена, назад, прогресс, вложенные flow
6. **[Темизация](theming.md)** — локализация всех кнопок и иконок

## Примеры

Рабочие боты, которые можно запустить сразу:

- **[quickstart.py](../../examples/quickstart.py)** — ~60 строк, один flow + один browse
- **[casino.py](../../examples/casino.py)** — все пять паттернов в одном файле

```bash
BOT_TOKEN=... uv run python examples/quickstart.py
```
